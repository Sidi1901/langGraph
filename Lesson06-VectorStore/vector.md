
A vector store is a database optimized for storing and searching embeddings.
Embedding = Text → Numbers


#### Key Features

Storage: Ensures that vectors and their emeddings are retained, whether in-memory or on-disk.

Similarity search : Helps retrieve the vectors most similar to a query

Indexing : Provides data structure or method that enables fast similarity searches on high-dimenional vectors.

CRUD ops : Manage lifecycle of vectors


Common Vector Stores : FAISS, Chroma, Pinecone, Weaviate
